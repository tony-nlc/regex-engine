# Regex Engine
This is a simple and basic project that allow you to you regex search and match without in your command line interface.

## Current support token:
- [x] Character
- [x] . # Any case
- [x] + # One or more
- [x] ? # Zero or one 
- [] ^  # Start of string
- [] $  # End of string
- [] \n # Newline
- [] \r # Carriage return
= [] [] # Single Character in brace
